Custom STM32 Sensor Interface Board

  • Documented best practices for bare IC design

Wireless Torsional Strain Measurement System

Designed lightweight, fully wireless strain measurement system for Baja SAE vehicle axles. Collaborative design with Drivetrain subsystem ensured perfect integration with existing vehicle architecture.

  • Wireless data transmission with local flash backup
  • Multiple auxiliary sensors for comprehensive data
  • Custom enclosure designed around axle dimensions
